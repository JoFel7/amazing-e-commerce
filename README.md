# Amazing E-Commerce

## Description
This works on the backend to add routes to work with the models. Being able to GET, POST, PUT, DELETE Products, Tags, and Categories.
  

  
## Installation

Install project with npm

```bash
  npm i
```
    
## Usage/Examples
npm run seed then npm start to star application.
